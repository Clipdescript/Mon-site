Mentions Légales

1. Éditeur et hébergement

Le site web Horloge Interactive est un projet communautaire open source, développé et maintenu par des contributeurs bénévoles. Il est hébergé sur https://github.com/Clipdescript/Mon-site via https://pages.github.com/.

2. Licence et contributions

Ce projet est publié sous licence open source. Le code source est disponible publiquement pour consultation, modification et distribution. Les contributions sont les bienvenues.

Les icônes utilisées proviennent de Material Icons et sont soumises à leur propre licence.

3. Données personnelles

Aucune donnée personnelle n'est collectée. Le site fonctionne entièrement dans votre navigateur. Les préférences (thème et volume) sont stockées localement via localStorage.
